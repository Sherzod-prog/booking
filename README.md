# 🏨 Booking System Project

Ushbu loyiha mehmonxonalar yoki xizmatlarni bron qilish uchun mo'ljallangan to'liq Full-stack ilovadir. Tizim foydalanuvchilar uchun mijoz qismi va administratorlar uchun boshqaruv panelini o'z ichiga oladi.

---

## 🛠 Texnologiyalar Arxivaturasi

Loyiha uchta asosiy qismdan iborat:

| Qism | Texnologiya |
| :--- | :--- | :--- |
| **Backend** | `NestJS` |
| **Frontend** | `React + Vite` |
| **Dashboard** | `React Admin` |
| **Database** | `PostgreSQL` |
